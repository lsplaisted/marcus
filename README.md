# M.A.R.C.U.S.

Multiple Amazing Robot Code Usage Selector
Making All Robots Capable of Usage Selection
Menu Activated Robot Code Usage System


# How to set up

- Copy M.A.R.C.U.S. code into your repo or a local folder
- Install uv
- Run `uv sync` from that folder
- Install VS Code extensions
  - Python
  - BlocklyPy
- Open code folder in VS Code
- Update robot.py for your robot
  - Set TIRE_DIAMETER and AXLE_TRACK appropriately
  - Modify the ports and directions of the different motors if needed
